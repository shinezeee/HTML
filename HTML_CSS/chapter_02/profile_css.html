Engine11 Vortex 2015

Engine11의 Vortex 2015년식 프레임.

Vortex의 첫번째 모델이다. 6061 더블버티드 알로이 프레임, 알카본 포크가 적용되었다.
첫 모델인 만큼 소량으로 생산되어 현재까지 잘 남아있지 않아 소중한 프레임이다.

2015년도 여름에 어렵게 완차로 구해 65만원 정도에 구매했다. (정가 73만원)
그때부터 하나하나 부품을 바꿔가며 지금의 개성있는 나만의 자전거가 완성 되었다.
이제는 나의 시그니쳐, 마스코트 자리 잡혔다.어쩌면 나의 아이덴디티 일지도..
